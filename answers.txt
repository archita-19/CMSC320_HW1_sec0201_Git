
Q1: Why might we consider using Git?

Answer: We might consider using Git because it allows developers to track changes and it allows us to collaborate with other developers in projects by working on local machines.

Q3: What is the difference between Merging and Rebasing?

Answer: The difference between merging and rebasing is that merges are made to apply changes from a feature branch to the current branch. Rebasing is used to apply changes from a branch and to also change the start of a branch to a newer commit.
